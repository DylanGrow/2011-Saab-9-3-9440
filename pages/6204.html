Position Sensor, Soft Top Lowered (562o)



Position Sensor, Soft Top Lowered (562o)

Location

Position sensor, soft top cover open (562o)






Main task

The sensor gives information to the STC control module indicating if the soft top cover is open.

Open






Closed






Type

Hall sensor with inbuilt magnet.

Technical data

Connection









The sensor is connected to the STC control module with two leads. The sensor is energized with B+ on pin 2 and gives information to the STC control module on pin 3 indicating if the soft top cover is open.

The sensor is affected by the cylinder piston. The sensor is affected when the soft top cover is open and the hydraulic piston is in its upper position.
