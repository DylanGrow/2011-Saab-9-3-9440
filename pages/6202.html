Position Sensor, Soft Top Cover Locked, LH (562LH)



Position Sensor, Soft Top Cover Locked, LH (562LH)

Location

Position sensor, soft top cover locked, LH (562LH)






Main task

The sensor informs the STC control module whether or not the soft top cover is locked on the LH side.

Unlocked






Locked






Type

Hall sensor with inbuilt magnet.

Technical data

Connection









The sensor is connected to the STC control module with two leads. The sensor is energized with B+ on pin 2 and gives information to the STC control module on pin 3.

The sensor is affected by a link on the LH side of the soft top cover. The sensor is affected when the soft top cover is locked.
