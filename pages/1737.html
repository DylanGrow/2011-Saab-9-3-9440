Locks: Testing and Inspection



The Central Locking System Locks/Unlocks When Driving Off

Background

During M05 the lock sensors were removed and instead "lock re-catch" was introduced which means that the locks always start by moving in the direction opposite to the one requested, e.g. to unlock, it first locks all doors and then unlocks. This is because the position of the locks is unknown.

Conditions

Cars with automatic locking when driving off. The function must be activated.

The following cars could be affected:

M05, 4D: VIN 51035621
M05, CV: VIN 56008018
M06: All
M07: All

Fault symptoms

The central locking system locks/unlocks several times when driving off (14 km/h).

Procedure

Update the BCM with the latest software.
