Position Sensor, Sixth Bow Locked, RH (578RH)



Position Sensor, Sixth Bow Locked, RH (578RH)

Location

Position sensor, sixth bow locked, RH (578RH)






Main task

The sensor informs the STC control module if the sixth bow is locked on the RH side.

Unlocked






Locked






Type

Hall sensor with inbuilt magnet.

Technical data

Connection









The sensor is connected to the STC control module with two leads. The sensor is energized with B+ on pin 2 and gives information to the STC control module on pin 3.

The sensor is affected by a metal tongue on the sixth bow mechanism. The sensor is affected when the sixth bow is locked on the RH side.
