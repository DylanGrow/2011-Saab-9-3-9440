Procedures

Procedure after disconnecting the battery

1. Clear diagnostic trouble codes in all systems.

Important
Some diagnostic trouble codes can be generated if there is a voltage drop. These diagnostic trouble codes can be ignored and will not be regenerated after deletion.

2. Set date and time.
3. Synchronize the remote control code by inserting the key into the ignition.

Note
When the CIM has been without a voltage supply, the car must be driven at a speed of at least 7 km/h (11 mph) and then stand still before the steering lock can function correctly.

Important
Draw the customer's attention to the fact that all remote controls must be synchronized by placing the key/keys in the ignition lock. If this is not done, the remote control will not work.

4. Cars with pinch protection: Program the pinch protection, see Programming pinch protection.

Warning
The trap protection is not active until the window lifters have been calibrated after a power failure.
