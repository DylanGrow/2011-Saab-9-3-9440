Service Indicator, Reset



Service Indicator, Reset

The menu is navigated via the steering wheel buttons and the information is displayed in SID.

1. Turn the ignition key to the ON position (engine should be off).
2. Select Settings using the INFO button. Hold the SET button depressed until it beeps.
3. Select Time for Service using the INFO button. Hold the SET button depressed until it beeps.
4. Select YES using the INFO button.
5. Cars with diesel engine: Hold the brake pedal depressed.
6. Reset the service indicator via a brief press on the SET button.
7. Turn the ignition key to the "LOCK" position.
