Settings, ACC System



Settings, ACC System

Settings

Some ACC system functions can be individually adapted.

1. Use the "Up" or "Down" steering wheel control to select "Settings".
2. Hold the SET button depressed until there is an audible signal.
3. Select "Climate system"
4. Do not press the SET button.
5. Select "Heated rear window", "Seat heating", "Fan speed", "A/C setting" or "Factory setting".
6. Press the SET button.
7. Make you desired choice.
8. End by pressing the SET button.

Use the CLR button to undo.

The following settings can be made:

Electrically heated rear window

AUTOMATIC

MANUAL

Seat heating

AUTOMATIC

MANUAL

Fan speed

HIGH

MEDIUM

LOW
