Transmission Speed Sensor: Description and Operation

Speed sensor, gearbox input shaft, FA 57 (532)

Location

Speed sensor, input shaft, transmission casing (532)

Main use

To measure the speed of the transmission input shaft. This value is used by TCM.





Type

Magnetoresistive, Hall effect. Detects rotation of 12-tooth wheel.

Activation

The sensor is powered from pin 32 (B) on TCM and the sensor signal is received on pin 22 (B) on TCM.
