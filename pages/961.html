Symptom Related Diagnostic Procedures

TC/ESP function cannot be switched off with shut-off button.

Fault symptom

TC/ESP function cannot be switched off with shut-off button.

Conditions

No diagnostic trouble codes stored in TCS/ESP.

Diagnostic help

Fault diagnosis indicates fault with just the ESP button with corresponding lead.

Button position can be read with the diagnostic tool.
