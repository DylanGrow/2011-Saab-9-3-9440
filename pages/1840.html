Soft Top Will Not Operate



Soft Top Will Not Operate

Background

If the boot lid is open when the soft top is operated using the remote control, it may render the soft top inoperable with the remote control as well as the switch. It is not necessary for the boot lid to be open fully, it is sufficient that the boot lid switch is depressed for the symptom to arise.

Fault symptoms

Soft top will not operate

Conditions

Cars as from VIN limit 46008086-
Open boot lid when operating with remote control.

Procedure

1. Lock the car.
2. Wait 30 seconds before unlocking the car.
