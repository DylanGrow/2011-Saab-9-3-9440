Main Fuse Box 701 In The Luggage Compartment (REC)



Main Fuse Box 701 In The Luggage Compartment (REC)

Location

The main fuse box is placed in the luggage compartment on the left-hand wheel housing.

Main function

To be the main fuse box for the rear of the car and roof and to take care of specific parts of the logic for e.g. rear lighting.

Type

Main fuse box with relays, maxi and micro fuses and integrated control modules for logic functions. The wiring harness from the trailer coupling is connected to the main fuse box.






Electrical connections

The main fuse box is fed with +30 voltage from the engine compartment's main fuse box 342 and is ground at grounding points G3 and G29. The main fuse box is connected to the wiring harness via 7 connectors:

1. Body 1 (in the wiring diagram = B1) to the main harness






2. Body 2 (in the wiring diagram = B2) to the main harness






3. Body 3 (in the wiring diagram = B3) to the main harness






4. Head 1 (in the wiring diagram = H1) to the roof harness






5. Head 2 (in the wiring diagram = H2)

4D/5D to the roof harness

CV to the main harness






6. Rear (in the wiring diagram = R) to the rear harness






7. Tail (in the wiring diagram = T)

4D: to the main harness






5D: to the rear harness






CV: to the rear harness






The connector for the main harness and rear harness consists of four sub-connectors, which are designated A, B, C and D, for example B1-B in the wiring diagram.

The contact TR-WH is used for the connection of the trailer contact. In some instance a special module needs to be connected to the TR-MOD socket.

The power circuits are distributed by the main fuse box:

Power supply +30, luggage compartment's main fuse box

Power supply +15, luggage compartment's main fuse box
